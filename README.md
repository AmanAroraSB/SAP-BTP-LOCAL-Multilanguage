# SAP CAP Localization Demo

## Overview
This project demonstrates how to implement **multi-language data support** in SAP CAP using:

- SAP CAP (Node.js)
- Localized entities (`localized`)
- Translation tables (`.texts`)
- Language handling via `Accept-Language`

---

##  Features

-  Product service (CAP)
-  Multi-language data (English + Hindi)
-  Automatic translation resolution
-  Clean separation of base data & translations
